Integrasi Testing

OPINI | 20 January 2011 | 16:24 Dibaca: 2039   Komentar: 0   0

Definisi :

pengujian Integrasi adalah tahap dalam pengujian perangkat lunak di mana modul perangkat lunak individu digabungkan dan diuji sebagai sebuah kelompok.

Tujuan :

  • pemeriksaan fungsional

  • Kinerja

  • kehandalan dari struktur program yang di rancang.
Beberapa jenis pengujian integrasi adalah :
-Pendekatan Big Bang
-Pendekatan Top-down, dan
-Pendekatan Bottom-down
Pendekatan Big Bang : Dalam pendekatan ini, semua atau sebagian besar dikembangkan modul yang digabungkan bersama untuk membentuk sebuah sistem perangkat lunak lengkap atau bagian utama dari sistem dan kemudian digunakan untuk pengujian integrasi. .
Integrasi Top-down : sebuah pendekatan untuk pengujian terpadu di mana modul terintegrasi atas diuji dan cabang modul diuji langkah demi langkah sampai akhir modul terkait.

terdapat 2 pola pengujian untuk top-down :

-

Bottom Up Pengujian adalah sebuah pendekatan untuk pengujian terpadu dimana komponen tingkat terendah diuji terlebih dahulu, kemudian digunakan untuk memfasilitasi pengujian komponen tingkat yang lebih tinggi.

Uji integarasi dalam konteks objek oriented dibagi menjadi 2 bagian yaitu :

1.thread base testing dilakukan dalam sekumpulan kelas yang dibutuhkan unutk menjawab satu input atau event untuk system, setiap thread di integrasi dan di uji individual.

2.usebase testing konstruksi system dimulai dengan menguji kelas-kelas yang menggunakan kelas server paling sedikit. Kelas ini disebut kelas independent, setelah kelas independent diuji, kelas yag menggunakannya diuji (kelas dependent). Pengujian kelas dependent dilakukan berturut-turut lapis demi lapis, hingga keseluruhan system dibangun.

Uji arsitektur client/server.Pengujian dilakukan terhadap aplikasi client, aplikasi server terkait dan arsitektur client server keseluruhan.levek yang digunakan dalam pengujian :

  • Aplikasi client individual diuji dalam mode disconnected dengan server.

  • Software client dan aplikasi server terkait diuji, tetapi operasi jaringan belum diperhatikan.

  • Arsitektur client server yang lengkap, termasuk operasi jaringan dan kinerjanya diuji

Uji integrasi memeriksa navigasi semantic, untuk meyakinkan setiap kebutuhan setiap kategori pengguna dapat dicapai.

Contoh :

Uji sintak navigasi, uji semantic navigasi.
